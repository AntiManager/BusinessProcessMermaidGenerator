🔧 РАБОЧИЙ ПРОЦЕСС

1. ЗАПУСК И ВЫБОР ДАННЫХ:
   - Запустите программу (автоматически открывается GUI)
   - Выберите файл Excel через кнопку "Обзор..."
   - ✅ АВТОМАТИЧЕСКИ загружается список доступных листов

2. ВЫБОР СИСТЕМЫ ВИЗУАЛИЗАЦИИ:
   - Бизнес-процессы: традиционный анализ операций
   - Causal Loop Diagrams: системная динамика

3. НАСТРОЙКА CLD (если выбран):
   - Выберите источник данных:
     * "Авто из бизнес-процессов" - автоматическое построение
     * "Из отдельного листа" - ручные данные CLD
   - Для ручного ввода выберите лист с CLD данными

4. ГЕНЕРАЦИЯ ДИАГРАММЫ:
   - Нажмите "🎯 Сгенерировать диаграмму"
   - Программа автоматически откроет результат в браузере
   - Для CLD доступны: статистика связей, петли обратной связи, реестр

ДЛЯ БИЗНЕС-ПРОЦЕССОВ (обязательные колонки):
┌──────────────┬─────────────┬─────────────────────┐
│ Колонка      │ Обязательно │ Описание           │
├──────────────┼─────────────┼─────────────────────┤
│ Операция     │ ✅ ДА       │ Название операции  │
│ Входы        │ ✅ ДА       │ Входные данные     │
│ Выход        │ ✅ ДА       │ Выходные данные    │
└──────────────┴─────────────┴─────────────────────┘

ДЛЯ CLD (ручной ввод):
┌──────────────┬─────────────┬─────────────────────┐
│ Колонка      │ Обязательно │ Описание           │
├──────────────┼─────────────┼─────────────────────┤
│ Источник     │ ✅ ДА       │ Исходная переменная│
│ Цель         │ ✅ ДА       │ Целевая переменная │
│ Знак влияния │ ✅ ДА       │ "+" или "-"        │
└──────────────┴─────────────┴─────────────────────┘

Опциональные колонки для обоих форматов:
┌────────────────────┬─────────────┬─────────────────────────┐
│ Колонка           │ Обязательно │ Описание               │
├────────────────────┼─────────────┼─────────────────────────┤
│ Группа            │ ❌ НЕТ      │ Группировка операций   │
│ Владелец          │ ❌ НЕТ      │ Ответственный           │
│ Подробное описание│ ❌ НЕТ      │ Детали операции        │
│ Сила влияния      │ ❌ НЕТ      │ Числовая оценка (CLD)  │
│ Учитывать в CLD   │ ❌ НЕТ      │ TRUE/FALSE для фильтра │
└────────────────────┴─────────────┴─────────────────────────┘

LEGEND - ЛЕГЕНДА ЦВЕТОВ И СИМВОЛОВ

БИЗНЕС-ПРОЦЕССЫ:
- 🟡 ЖЕЛТЫЕ ОВАЛЫ - внешние входы
- 🔴 КРАСНЫЕ ОВАЛЫ - конечные выходы  
- 🟠 ОРАНЖЕВЫЕ - операции слияния
- 🟣 ФИОЛЕТОВЫЕ - операции разветвления
- ❇️ ПУЛЬСИРУЮЩИЕ КРАСНЫЕ - супер-критические операции
- 🔵 СИНИЕ - стандартные операции

CAUSAL LOOP DIAGRAMS:
- 🔵 СИНИЕ ОВАЛЫ - переменные системы
- 🟢 ЗЕЛЕНЫЕ СТРЕЛКИ - положительное влияние (+)
- 🔴 КРАСНЫЕ СТРЕЛКИ - отрицательное влияние (-)
- 🔄 ПЕТЛИ - обнаруженные циклы обратной связи

АВТОСОХРАНЕНИЕ НАСТРОЕК:

- Восстановление предыдущей сессии при запуске

НОВЫЕ НАСТРОЙКИ CLD:
- cld_source_type: auto/manual (источник данных)
- cld_sheet_name: лист для CLD данных
- show_cld_operations: показывать операции на связях
- cld_influence_signs: показывать знаки влияния

НАСТРОЙКИ АНАЛИТИКИ:
- critical_min_inputs: порог для критических операций
- critical_min_reuse: минимальное использование выхода
- subgroup_column: колонка для группировки
- show_detailed: подробное описание в узлах

🐛 РЕШЕНИЕ ТИПИЧНЫХ ПРОБЛЕМ

ПРОБЛЕМА: Не отображаются связи в CLD
РЕШЕНИЕ: Проверьте обязательные колонки "Источник", "Цель", "Знак влияния"

ПРОБЛЕМА: Знаки влияния не отображаются в Mermaid
РЕШЕНИЕ: Mermaid автоматически экранирует специальные символы - используйте интерактивный формат

ПРОБЛЕМА: Петли обратной связи не обнаружены
РЕШЕНИЕ: Убедитесь что связи образуют замкнутые циклы

ПРОБЛЕМА: Данные не загружаются
РЕШЕНИЕ: Проверьте что выбран правильный лист и формат данных соответствует требованиям

ПРОБЛЕМА: Диаграмма не открывается в браузере
РЕШЕНИЕ: Проверьте путь к файлу - не должно быть кириллических символов
